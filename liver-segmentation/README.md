# Сегментация печени
## Цель  
Определить рак печени и локализовать его  
## Библиотеки  
- PyTorch
- Monai
- Numpy
- Matplotlib
- Nibabel
- Glob
- Albumentations
## Главная метрика оценки качества модели  
Индекс Жакара (реализован через логические операции)  
## Дополнительные метрики оценки качества модели  
Precision, Recall, F1-score
